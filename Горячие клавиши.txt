PYCHARM:
Ctrl + Shift + F10 — запустить текущий файл
Alt + Insert — создать файл/папку
Shift + F6 — переименовать файл или переменную
Ctrl + Q — посмотреть документацию по методу

Ctrl + D — дублировать текущую строчку кода
Ctrl + Y — удалить текущую строчку кода
Ctrl + Alt + L — форматировать код (!)
Ctrl + Z — отменить предыдущее действие
Ctrl + Shift + Z — отменить отмену
Ctrl + F — поиск по тексту (по коду или по выводу)

Alt + Shift + левая кнопка мыши — выделить несколько строк кода
Alt + Shift + вверх/вниз — поднять текущую строчку вверх/вниз

Ctrl + / — комментировать/раскомментировать

Ctrl + Alt + V — вынести в переменную выражение

WINDOWS:
Ctrl + C — копировать
Ctrl + V — вставить
Ctrl + Shift + N — создать новую папку

IDLE:
Alt + P — повторить предыдущее выражение
F5 — запустить текущий файлик
